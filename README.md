# Refuge for Humans — application

Code for [Refuge for Humans](https://github.com/c4g-john/refuge-for-humans-pmo).
Scope is governed by the documents in the PMO repo; delivery is tracked here as
Features and Stories scaffolded by the docassert bridge from the approved user
stories. Pull requests close stories with `Closes #N`; a Feature closes when
its last story lands.

- Scope (authoritative): the [PMO repo](https://github.com/c4g-john/refuge-for-humans-pmo)
- Status dashboard: https://c4g-john.github.io/refuge-for-humans-pmo/
